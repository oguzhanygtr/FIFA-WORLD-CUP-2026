# 🏆 2026 FIFA WORLD CUP - Data Analysis

⚽ **Discover the future of soccer through data!**  
This project aims to analyze match data from the 2026 FIFA World Cup to uncover team performance, player statistics, and tournament trends.  
We’re here to gain a deeper understanding of soccer using modern data science tools.

- 🎨 Data visualizations (charts, tables, dashboards)
